Place future Shopify Buy Button HTML snippets here.

Use the slug filenames from `src/data/shopCollections.ts`, for example:

- `blue-archive.html`
- `monster-hunter.html`
- `project-sekai.html`

The shop page will try to load the matching file when its filter button is clicked.

The default `全部` collection is configured directly in `src/data/shopCollections.ts` with the live Shopify collection ID.
